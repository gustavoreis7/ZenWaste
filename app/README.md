♻️ ZenWaste

Plataforma web B2B para gestão de resíduos industriais, marketplace e inteligência de mercado.

🚀 Sobre o Projeto

O ZenWaste é um sistema desenvolvido para ajudar empresas a gerenciar seus resíduos de forma eficiente, organizada e alinhada com boas práticas ambientais.

A plataforma permite o controle de estoque de resíduos, cadastro de empresas e publicação de anúncios, criando um ecossistema que conecta geradores e possíveis compradores.

🎯 Objetivo
Digitalizar a gestão de resíduos industriais
Facilitar a rastreabilidade e controle de estoque
Criar um marketplace para reutilização de resíduos
Gerar inteligência de mercado a partir dos dados
🧩 Funcionalidades

✔️ Cadastro de empresas
✔️ Gestão de estoque de resíduos
✔️ CRUD completo de produtos
✔️ Publicação de anúncios
✔️ API REST em JSON
✔️ Estrutura preparada para expansão (marketplace + analytics)

🏗️ Arquitetura
Backend
Framework: Django
API: Django REST Framework
Banco de dados: SQLite (ambiente de desenvolvimento)
